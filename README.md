# Movie Recommendation System 🎬

## Overview
A Machine Learning based Movie Recommendation
System that suggests similar movies to users
based on content-based filtering using
genres and tags.

## Dataset
MovieLens Dataset
- Movies dataset (movies.csv)
- Ratings dataset (ratings.csv)

## Model Used
Content-Based Filtering
- TF-IDF Vectorizer
- Cosine Similarity

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Steps
1. Data Loading
2. Data Preprocessing
3. Feature Extraction (TF-IDF)
4. Cosine Similarity Calculation
5. Movie Recommendation
6. User Input Prediction

## How to Use
1. Run all cells in order
2. Enter a movie name when prompted
3. Get top 5 similar movie recommendations

## Sample Output
Enter movie name: Toy Story
Top 5 Recommendations:
1. Toy Story 2
2. A Bug's Life
3. Monsters, Inc.
4. The Incredibles
5. Finding Nemo
